# CONAMFI - Integral Management Platform for National Academic Congress

> **Full-stack platform** for participant registration, academic paper submissions, workshop management, and real-time event administration for the National Marist Congress.

---

## Overview

**CONAMFI** is an integral management platform developed for the National Marist Congress of Integral Formation (Congreso Nacional Marista de Formacion Integral). Built under a hybrid model, it enables seamless management for both in-person speakers and virtual attendees, supporting multi-modal registration, academic poster submissions across thematic axes, and automated workshop capacity control with real-time admin dashboards.

### Key Features

| Area | Capabilities |
|------|-------------|
| **Multi-Modal Registration** | Support for in-person and virtual attendees with institution and role capture |
| **Academic Submissions** | Poster submission module with thematic axes (AI in Education, Academic Integrity, etc.) |
| **Workshop Management** | Automatic capacity control, virtual access links, and session scheduling |
| **Admin Dashboard** | Real-time user monitoring, connection status, and role-based permissions |
| **Data Export** | Attendance reports and professional spreadsheet generation for external analysis |
| **Automated Notifications** | Email system for confirmations, updates, and event communications |

## Tech Stack

### Frontend
| Technology | Use |
|------------|-----|
| **Next.js** | Full-stack React framework with instant navigation |
| **React** | Component-based interactive UI |
| **Tailwind CSS** | Responsive design with mobile-first approach |
| **shadcn/ui** | Professional UI component library |
| **Recharts** | Statistical data visualization in admin panel |

### Backend
| Technology | Use |
|------------|-----|
| **Next.js API** | Server-side API routes and SSR |
| **Prisma** | Type-safe ORM for database operations |
| **PostgreSQL** | Relational database for participant and event data |
| **NextAuth** | Authentication with encrypted sessions and role management |
| **AWS S3** | Secure storage for academic papers and posters |
| **Resend** | Automated professional email delivery |
