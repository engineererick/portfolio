# CONAMFI - Plataforma Integral de Gestión para Congreso Académico Nacional

> **Plataforma full-stack** para registro de participantes, envío de trabajos académicos, gestión de talleres y administración de eventos en tiempo real para el Congreso Nacional Marista.

---

## Visión General

**CONAMFI** es una plataforma integral de gestión desarrollada para el Congreso Nacional Marista de Formación Integral. Construida bajo un modelo híbrido, permite la gestión fluida tanto para ponentes presenciales como para asistentes virtuales, soportando registro multimodal, envío de carteles académicos a través de ejes temáticos y control automatizado de capacidad de talleres con dashboards de administración en tiempo real.

### Características Principales

| Área | Capacidades |
|------|-------------|
| **Registro Multimodal** | Soporte para asistentes presenciales y virtuales con captura de institución y rol |
| **Envío de Trabajos Académicos** | Módulo de envío de carteles con ejes temáticos (IA en Educación, Integridad Académica, etc.) |
| **Gestión de Talleres** | Control automático de capacidad, enlaces de acceso virtual y programación de sesiones |
| **Dashboard de Administración** | Monitoreo de usuarios en tiempo real, estado de conexión y permisos basados en roles |
| **Exportación de Datos** | Reportes de asistencia y generación de hojas de cálculo profesionales para análisis externo |
| **Notificaciones Automatizadas** | Sistema de correo para confirmaciones, actualizaciones y comunicaciones del evento |

## Stack Tecnológico

### Frontend
| Tecnología | Uso |
|------------|-----|
| **Next.js** | Framework React full-stack con navegación instantánea |
| **React** | UI interactiva basada en componentes |
| **Tailwind CSS** | Diseño responsive con enfoque mobile-first |
| **shadcn/ui** | Librería de componentes UI profesional |
| **Recharts** | Visualización estadística de datos en panel de administración |

### Backend
| Tecnología | Uso |
|------------|-----|
| **Next.js API** | Rutas API del lado del servidor y SSR |
| **Prisma** | ORM con tipado seguro para operaciones de base de datos |
| **PostgreSQL** | Base de datos relacional para datos de participantes y eventos |
| **NextAuth** | Autenticación con sesiones cifradas y gestión de roles |
| **AWS S3** | Almacenamiento seguro de trabajos académicos y carteles |
| **Resend** | Envío de correos profesionales automatizados |
