# Turny - eCommerce Returns & Marketplace Management Platform

> **Full-stack eCommerce platform** for order management, returns automation, and marketplace integration across Shopify, Magento, Vtex, and Mirakl.

---

## Overview

**Turny** is an enterprise-grade order portal designed for eCommerce that integrates platforms such as Shopify, Magento and Vtex, and facilitates returns management for Marketplaces using Mirakl API. The system reduced dispute processing time by 80% and manages over 100,000 returns per cycle with real-time synchronization across multiple eCommerce platforms.

### Key Features

| Area | Capabilities |
|------|-------------|
| **Multi-Platform Integration** | Shopify, Magento, Vtex, and Mirakl API with OAuth plugin connections |
| **Returns Management** | Centralized dashboard for claims, automatic PDF generation, status tracking |
| **Real-Time Sync** | Transaction-based synchronization handling 100,000+ returns per cycle |
| **Marketplace Analytics** | Sales dashboards with $94M+ tracked, refund rates, SKU-level metrics |
| **Customizable UX** | White-label system for emails, UI texts, logos, and payment links per client |
| **Upselling Module** | Banner and product management for cross-selling within the returns portal |
