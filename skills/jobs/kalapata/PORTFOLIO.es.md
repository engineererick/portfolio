# Turny - Plataforma de Devoluciones eCommerce y Gestión de Marketplaces

> **Plataforma eCommerce full-stack** para gestión de órdenes, automatización de devoluciones e integración de marketplaces con Shopify, Magento, Vtex y Mirakl.

---

## Visión General

**Turny** es un portal de órdenes de grado empresarial diseñado para eCommerce que integra plataformas como Shopify, Magento y Vtex, y facilita la gestión de devoluciones para Marketplaces usando la API de Mirakl. El sistema redujo el tiempo de procesamiento de disputas en un 80% y gestiona más de 100,000 devoluciones por ciclo con sincronización en tiempo real a través de múltiples plataformas de eCommerce.

### Características Principales

| Área | Capacidades |
|------|-------------|
| **Integración Multi-Plataforma** | Shopify, Magento, Vtex y Mirakl API con conexiones de plugins OAuth |
| **Gestión de Devoluciones** | Dashboard centralizado para reclamaciones, generación automática de PDF, seguimiento de estado |
| **Sincronización en Tiempo Real** | Sincronización basada en transacciones manejando 100,000+ devoluciones por ciclo |
| **Analítica de Marketplace** | Dashboards de ventas con $94M+ rastreados, tasas de reembolso, métricas a nivel SKU |
| **UX Personalizable** | Sistema white-label para correos, textos UI, logos y links de pago por cliente |
| **Módulo de Upselling** | Gestión de banners y productos para venta cruzada dentro del portal de devoluciones |

## Stack Tecnológico

### Frontend
| Tecnología | Uso |
|------------|-----|
| **PHP** | Renderizado del lado del servidor y plantillas |
| **JavaScript** | Dashboards interactivos y gráficos |
| **jQuery** | Manipulación DOM y solicitudes AJAX |
| **CSS** | Diseño responsive con temas personalizados |

### Backend
| Tecnología | Uso |
|------------|-----|
| **PHP 8** | Lógica de negocio central y capa API |
| **MySQL** | Base de datos relacional vía AWS RDS |
| **AWS EC2** | Infraestructura de cómputo en la nube |
| **AWS S3** | Almacenamiento de documentos y archivos |
| **Shopify API** | Órdenes eCommerce e integración OAuth |
| **Magento API** | Conector de plataforma eCommerce |
| **Vtex API** | Conector de plataforma eCommerce |
| **Mirakl API** | Gestión de devoluciones de marketplace |
| **Third-Party APIs** | Links de pago, logística y servicios de notificación |
