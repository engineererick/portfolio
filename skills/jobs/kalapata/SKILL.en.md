# Role: Software Engineer at Kalapata
**Timeline**: June 2022 - July 2024

## Context
Kalapata involved building complex full-stack platforms for eCommerce, real estate and logistics. A major highlight was the Turny platform, automating marketplace operations for Tier 1 retailers like Liverpool, Coppel, and Mercado Libre.

## Key Projects:
- **Turny (eCommerce Returns Portal)**: Full-stack order/returns platform integrating Shopify, Magento, Vtex, and Mirakl API. Built with PHP 8 + AWS (EC2, S3, RDS). Managed 100,000+ returns per cycle with real-time sync. 80% reduction in dispute processing time.
- **Segmail (Delivery Tracking)**: Real-time GPS tracking web app built with React/Node.js (TypeScript) + OnFleet API. Interactive maps with live progress bars and delivery status.
- **eHubs (Real Estate Platform)**: Property management platform with React/Node.js (TypeScript) + Google Maps API. Features include GIS clustering, automated PDF proposals, and advanced filtering.
- **AI Recommendation Engine**: LLM-based system for Segment of Totalplay.
