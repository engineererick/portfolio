# Role: Ingeniero de Software at Kalapata
**Timeline**: Junio 2022 - Julio 2024

## Context
Kalapata es una empresa de sociedad de cartera. Desarrollé plataformas full-stack complejas para eCommerce, bienes raíces y logística. Un punto destacado fue la plataforma Turny, automatizando operaciones de marketplace para retailers de primer nivel como Liverpool, Coppel y Mercado Libre.

## Key Projects:
- **Turny (Portal de Devoluciones eCommerce)**: Plataforma full-stack de órdenes/devoluciones integrando Shopify, Magento, Vtex y Mirakl API. Construida con PHP 8 + AWS (EC2, S3, RDS). Gestionó más de 100,000 devoluciones por ciclo con sincronización en tiempo real. Reducción del 80% en tiempo de procesamiento de disputas.
- **Segmail (Rastreo de Entregas)**: Aplicación web de rastreo GPS en tiempo real construida con React/Node.js (TypeScript) + OnFleet API. Mapas interactivos con barras de progreso en vivo y estado de entrega.
- **eHubs (Plataforma de Bienes Raíces)**: Plataforma de gestión de propiedades con React/Node.js (TypeScript) + Google Maps API. Incluye clustering GIS, propuestas PDF automatizadas y filtrado avanzado.
- **Motor de Recomendación IA**: Sistema basado en LLM para Segmento de Totalplay.
